Core Capabilities

What Dagens Provides

settings_input_component

Distributed Control Plane

Centralized management for durable recovery and lifecycle state. Dagens ensures every workflow transition is recorded and recoverable across your entire cluster.

Architecture Overview open_in_new
account_tree

Worker Fabric and Horizontal Execution

Capacity-aware dispatch across workers lets execution scale horizontally while the control plane remains the durable source of truth for workflow progress.

security

HA Failover and Dispatch Fencing

Leadership-gated dispatch prevents duplicate execution. Automatically recover workflows on healthy nodes if a worker or network partition occurs.

Human-in-the-Loop

Pause execution at critical points, collect human input through callbacks, and resume with durable checkpoint and recovery semantics.

Infrastructure Excellence

Architecture: Control Plane + Execution Plane

By decoupling scheduling from execution, Dagens keeps durable transition history in the control plane while workers remain execution units. This supports replay, recovery, and controlled failover when infrastructure fails.

history

Durable Transition History

Lifecycle state is persisted so workflows can be replayed and recovered after failure or restart.

sync_alt

State Decoupling

Workers are stateless execution units; the control plane holds the source of truth for all workflow progress.

shield

Leader-Gated Reliability

Prevent duplicate execution with strict dispatch fencing and leader election protocols.
